# Rocketeer
* php
* similar to Capistrano - written to not have ruby files in a php project
* requires Mcrypt and Composer be installed on remote server  


```
| .rocketeer  
| |-- config.php  
| |-- hooks.php  
| |-- paths.php  
| |-- remote.php  
| |-- scm.php  
| |-- stages.php  
| |-- events.php  
| |-- tasks.php  
```
